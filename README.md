# 📈 stockLog

주식 매매 기록 · 분석 · 전략 관리 · 커뮤니티 기능을 하나로 통합한 **올인원 트레이딩 로그 앱**입니다.

---

# 🧾 프로젝트 소개

**stockLog**는 개인 투자자가 매매 기록을 체계적으로 관리하고, 손익을 시각적으로 분석하며, 투자 전략을 점검하고, 다른 투자자와 인사이트를 공유할 수 있도록 제작된 앱입니다.

기존 노션 · 엑셀 기반 매매일지의 불편함을 개선하고, 증권사 API 및 외부 데이터 연동을 통해 자동화된 투자 관리 환경을 제공합니다.

# 📝 매매일지 기능

캘린더 기반 매매 기록 관리

## ✔ 작성 방법

* 캘린더에서 날짜 선택
* 선택한 날짜 자동 기입
* 매매일지 작성 가능

## ✔ 입력 항목

* 거래 날짜 (자동 입력)
* 국내 / 해외 주식 선택
* 종목명
* 증권사 선택
* 거래 종류 (매수 / 매도)
* 체결 단가
* 체결 수량
* 매매 이유
* 차트 이미지 첨부
* 해시태그 선택
* 매매 비용

👉 **예상 총 매수 금액 자동 계산**

## ✔ 캘린더 시각화

* 매수 / 매도 여부에 따라
* 날짜별 색상 점 표시
